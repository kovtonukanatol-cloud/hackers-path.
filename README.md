# Hacker's Path

A 2D hacker simulator set in a cyberpunk world. Built with Python and Pygame, designed for Linux.

You are a freelance hacker. Manage your balance, stay anonymous, upgrade your hardware — 
and don't get caught.

## Gameplay

- **Terminal UI** — visual style mimics a real Linux console
- **Hacking mini-game** — timing and reaction-based system
- **Resource management** — balance money, anonymity and hardware upgrades
- **Black Market** — buy upgrades, tools and stay ahead
- **Random events** — unpredictable situations that change the game

## The Fourth Wall Feature

The game reads real data from your system — your username, IP address — 
to make the experience feel disturbingly real.

If your **exposure reaches 100%**, the arrest sequence triggers:
- A hidden log file is created on your actual computer
- The game force-closes, simulating equipment seizure
- There is no save. There is no escape.

## Built With

- Python 3
- Pygame
- Linux (recommended)
